# farm-localstack-template

## Description

This template creates a fast and simple Localstack environment for development.

Dependencies:
 - [uv](https://github.com/astral-sh/uv)
 - [Docker (with Docker Compose)](https://www.docker.com/)

Use:
```sh
git clone "This repo"
docker compose up -d
uv install
uv run main.py
```
